### 1. Clone the repository

Backend

```bash
cd backend
npm install
```

Frontend

```bash
cd frontend
npm install
```

## Running the Application

Start the Backend Server

```bash
cd backend
npm run dev
```

The backend server should start (typically on port 3001 or as configured).

Start the Frontend Application

```bash
cd frontend
npm start
```
